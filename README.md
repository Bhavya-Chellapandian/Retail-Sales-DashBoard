📊 Retail Sales Data Analysis Using Excel

Project Overview

This project focuses on analyzing retail sales data using Microsoft Excel to generate meaningful business insights through data cleaning, transformation, analysis, and visualization. The project demonstrates essential Excel skills required for a Data Analyst role. 

Objectives

The project aims to answer key business questions such as:

* Which products generate the highest revenue?
* Which regions perform best?
* Which salespersons contribute the most to sales?
* What are the monthly sales trends?
* How do product categories perform?
* What are the customer purchasing patterns?

Key Business Insights

1. South Region generated the highest revenue.
2. Electronics is the top-performing category.
3. Top 20% of salespersons contribute nearly 60% of total revenue.
4. Sales peak during festive seasons.
5. Some high-revenue products have lower profit margins, indicating pricing optimization opportunities.
